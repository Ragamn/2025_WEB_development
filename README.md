# 2025_WEB_development
# Webサイト制作プロジェクト 開発ドキュメント

このリポジトリは、チームでWebサイトを構築するためのソースコードを管理します。
開発をスムーズに進めるため、以下のルールに従って作業をお願いします。

## 📁 ディレクトリ構成（フォルダ構成）

**「共通ファイル」と「ページ専用ファイル」を明確に分けています。**
新規ページを作成する際は、この構成を守ってください。

```text
root/
├── index.html            # トップページ
├── about/                # [例] 会社概要カテゴリ
│   └── index.html        # 会社概要ページ本体
├── contact/              # [例] お問い合わせカテゴリ
│   └── index.html        # お問い合わせページ本体
│
├── css/
│   ├── destyle.css       # リセットCSS（編集禁止）
│   ├── common.css        # 全ページ共通（ヘッダー、フッター、共通パーツ）
│   └── pages/            # ★ページごとの固有CSSはここ
│       ├── top.css       # トップページ用
│       ├── about.css     # 会社概要用
│       └── contact.css   # お問い合わせ用
│
├── js/
│   └── pages/            # ★ページごとの固有JSはここ
│       ├── top.js        # トップページ用（スライダー等）
│       └── contact.js    # お問い合わせ用（バリデーション等）
│
├── images/
    ├── common/           # ロゴ、共通アイコンなど
    └── about/            # そのページでしか使わない画像など
